## Run Locally

Clone the project

```bash
  git clone git@github.com:Wintrack/Workshop_MERN_Stack.git
```

Go to the project directory

```bash
  cd Workshop_MERN_Stack
```

Go to the front or back directory

```bash
  cd frontend
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
